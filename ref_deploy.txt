## 🔴 ROOT CAUSE (clear and simple)

You are using **Vite** (`vite.config.ts` exists), **but your `index.html` does NOT load your React entry file at all**.

👉 There is **NO `<script type="module">`** in your `index.html`.

So:

* Browser loads HTML ✅
* No JS bundle is loaded ❌
* React never mounts ❌
* Result: **blank white page**

This is why you see only favicon / asset 404s and nothing else.

---

## ✅ Why this happened (Google AI Studio quirk)

Google AI Studio “Build” sometimes:

* Generates **ESM importmap–style HTML**
* Assumes a **cloud runtime**
* Does **not wire Vite entry automatically**

Locally, Vite **requires** an explicit module entry.

## ⚠️ VERY IMPORTANT: Import Map Warning

You currently have this in `index.html`:

```html
<script type="importmap">
```

### 🚨 This is NOT how Vite works

Vite:

* Uses **node_modules**
* Bundles dependencies
* **Ignores import maps**

👉 This import map is **harmless but useless** locally.

**Do NOT remove it now**, but:

* Do NOT rely on it
* Ensure dependencies exist in `package.json`
